# CS50 AI Projects

This repository contains my completed projects from **Harvard's CS50 Introduction to Artificial Intelligence with Python**.

Throughout the course, I implemented algorithms and AI systems spanning graph search, game theory, logical reasoning, probabilistic inference, machine learning, neural networks, and natural language processing. Each project focuses on a different area of artificial intelligence and demonstrates practical problem-solving in Python.

## Projects

### Degrees

Uses graph search to determine the shortest connection between two actors based on the movies they've appeared in together.

### Tic-Tac-Toe

Implements an optimal Tic-Tac-Toe AI using the Minimax algorithm, ensuring the computer always makes the best possible move.

### Knights

Solves Knights and Knaves logic puzzles through propositional logic and automated logical inference.

### Minesweeper

Builds an AI capable of playing Minesweeper by making logical deductions from known information about the board.

### PageRank

Implements Google's PageRank algorithm using both sampling and iterative methods to rank web pages by importance.

### Heredity

Uses Bayesian networks to calculate the probability that individuals inherit specific genes and traits.

### Crossword

Generates crossword puzzle solutions using constraint satisfaction, backtracking search, and inference techniques.

### Shopping

Develops a machine learning classifier with scikit-learn to predict whether an online customer will complete a purchase.

### Nim

Trains an AI agent to play the game of Nim using reinforcement learning (Q-learning).

### Traffic

Builds a convolutional neural network (CNN) with TensorFlow to classify traffic signs from image data.

### Parser

Creates a natural language parser using context-free grammar to identify the grammatical structure of English sentences.

### Questions

Implements a question-answering system that retrieves the most relevant passages and sentences from a collection of documents using natural language processing.
